About

We are a small team of enthusiasts passionate about .NET, developer tools and code search.

Ref12 is a powerful new engine inspired by source.dot.net and referencesource.microsoft.com. It is an experimental replacement/successor to sites generated by SourceBrowser. Ref12 allows searching and navigating across repositories along with incremental addition of new repositories.

Our goal is to sustainably grow adoption of semantic code search and navigation for open source projects in the .NET ecosystem.

  • This site is not provided by Microsoft or GitHub.
  • The new engine is currently not open-source.
  • Currently only public GitHub repositories can be indexed.
  • Append org/repo to scope to a specific repository: ref12.io/dotnet/roslyn
